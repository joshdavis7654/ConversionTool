# ConversionTool
A simple web tool to convert between units using javascript. Currently can convert types of weight and length. In the process of adding currency based on current dollar values. 

Weight types:
Pounds
Kilograms
Stone
Newtons
Grams
Ounces

Length types:
Meters
Millimeters
Centimeters
Feet
Inches
Yards
Miles

When user selects a conversion type lists will populate. Info will appear at the bottom if user has not selected proper conversion type or units to convert. Will warn user when a non-valid number has been inserted. Units are converted as the user inputs.
